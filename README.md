# Specs
* ubuntu:18.04
* PHP 7.1
* NGINX ( serves PORT 80 )
* Composer
* Image Magick
* Memcached
* Support Laravel 5

# F.A.Q
* Remember to add any system service in sh/start-services.sh and set it as ENTRYPOINT to start them when container starts
* This image should be use as a base image for others
